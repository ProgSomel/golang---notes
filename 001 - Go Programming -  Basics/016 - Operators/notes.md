# Operators

## Logical Operators:
- ! (Logical NOT)
- || (Logical OR)
- && (Logical AND)

## Bitwise Operators:
- & (Bitwise AND)
- | (Bitwise OR)
- ^ (Bitwise XOR)
- &^ (Bitwise AND NOT)
- << (Left shift)
- >> (Right shift)

## Comparison Operators:
- == (Equal)
- != (Not Equal)
- < (Less than)
- <= (Less than or Equal to)
- > (Greater than )
- >= (Greater than or Equal to)
